Project Overview

This project is a website design of a functional and user-friendly Library Management system developed using Flask|HTML|CSS|JAVA.

It demonstrates the connection between frontend and backend development and communication between them.

The focus is on usability, smooth dataflow, and a clean design.

Additional Details

To run the code a few dependencies need to be installed and the Flask server must be started.

To start off active the enviroment with the following command:

.\venv\Scripts\Activate.ps1

Then install Flask within the enviroment with:

pip install Flask

Finally run the app.py file and the program should be working
